Add Confluence search directly to your Chrome address bar. Type a keyword (like con) and search instantly!

Installation Steps

Step 1: Chrome should detect the search engine automatically when you visit this page. Look for a prompt in the address bar.
Step 2: If no prompt appears, go to chrome://settings/searchEngines
Step 3: Click "Add" under "Site search" and enter:
  • Search engine: Confluence
  • Keyword: con (or whatever you prefer)
  • URL: https://your-confluence.atlassian.net/wiki/search?text=%s
Step 4: Test it! Type con your search query in Chrome's address bar
